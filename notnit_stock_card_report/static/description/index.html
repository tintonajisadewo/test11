Kartu Stok (Stock Card)

- Print Kartu Stok (Stock Card) of some Product on spesific Warehouse location and periode

- Show Specific Date Range

- Output Report in PDF or Excel (XLSX) file format

How to use:

1. Go to Inventory

Go to Report klick Stock Card

2. Choose : Period ,Start Date, End Date,Product, Location

Click Action Button then click EXPORT PDF.

3. Print Report

3a. Click Export PDF, then Kartu Stok (Stock Card) Report will appear in PDF format file.

3b. Click Export Excel, then Kartu Stok (Stock Card) Report will appear in Excel (XLSX) format file.
